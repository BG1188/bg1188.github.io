TikTok创建Pixel - Shopify

TikTok注册BC

浏览器打开网址https://business.tiktok.com/zh，点击右上角立即注册 输入注册邮箱和密码并同意条款，点击下方注册 填写邮箱收到的验证码 由于电脑使用节点的差异下一步会分两种情况（比如使用香港台湾节点会显示情况一，美国节点会显示情况二） 情况一：新建商务中心，输入商务中心名称（随便写，建议用公司名称或者品牌名称）和 时区（随便选，建议UCT+08），头像随意设置，然后点击创建 下一步到这个页面表明BC创建已完成，红框标注的地方就是BC ID 情况二：选择我是广告主，地区 公司名称 行业 币种随便写，商务中心名称（随便写，建议用公司名称或者品牌名称）和 时区（随便选，建议UCT+08），最后点左下方的跳过按钮，不要点创建 点击跳过之后，选择创建基本账号 下一步到这个页面表明BC创建已完成，红框标注的地方就是BC ID 如果已有BC，想注册第二个或者第三个BC，看下面步骤 打开链接https://business.tiktok.com/select，点击右上角新建账号 接下来步骤跟上面第一次创建步骤一致
